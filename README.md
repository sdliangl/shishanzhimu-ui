# 狮山智牧 UI设计展示

> 数字孪生视域下农村产业扶贫的"游戏化"逻辑与信任存证研究及产业化应用

## 项目简介

狮山智牧是一个基于北斗定位、区块链溯源的智慧认养禽畜小程序，旨在通过游戏化养成体验，让城市用户远程认养农村散养禽畜，实现乡村振兴与数字农业的创新融合。

## 核心功能

- 🎮 **游戏化养成**：虚拟宠物养成体验，每日任务互动
- 📍 **北斗定位**：实时追踪禽畜GPS轨迹，活动范围可视化
- ⛓️ **区块链溯源**：完整的信任存证体系，每条记录上链
- 👨‍🌾 **农户增收**：溢价150%，助力乡村振兴
- 🏆 **社区互动**：排行榜、动态分享、积分系统

## 技术栈

- **前端**：uni-app（跨平台小程序）
- **后端**：Node.js + Express
- **数据库**：MySQL + Redis
- **区块链**：长安链（ChainMaker）
- **定位**：北斗三号（UC6226芯片）
- **通信**：MQTT + NB-IoT
